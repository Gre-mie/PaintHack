# PaintHack
A simple drawing program made for a Hackathon

- Right mouse button - draw to the canvas

## Debug mode

Debug mode is a tool to see how things are drawn.

WARNING: exiting debug mode will clear the canvas

- d - enter/exit debug mode
- t - test mode (does nothing)
- l - line mode - shows how a line is drawn, click two points then click again to show the points drawn between

## Known Bugs

- Debug mode indicator doesn't show when hovering over canvas
